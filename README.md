# 터미널

## 소개
**터미널**은 개발자분들을 위한 스터디 모임 앱 서비스입니다.

혼자하는 개발 공부 힘들었다면?<br>
터미널에서 함께 공부할 사람들을 찾아보세요!<br>
서로 응원하고 힘을 합치면 원하는 목표를 달성할 수 있을 거예요.
